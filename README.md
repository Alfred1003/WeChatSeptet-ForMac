## 简版特色功能，多开时使用不同主题
- 多开了微信A与微信B，假设此时都是`黑夜模式`。
- 在微信B中选择`少女模式`，微信B退出。
- 在微信A中点击多开，登录微信B，即可实现两个微信使用不同的皮肤。
